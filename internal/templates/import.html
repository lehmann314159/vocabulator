Import Words

Upload a CSV file to import words in bulk

CSV Format

The CSV file should have the following columns:

  • word (required) - The vocabulary word
  • source (required) - Where you learned it
  • date_learned (required) - Date in YYYY-MM-DD format
  • part_of_speech (optional) - noun, verb, adjective, etc.
  • example_sentence (optional) - Usage example
  • tags (optional) - Comma-separated tags

Example:

word,source,date_learned,part_of_speech,example_sentence,tags
ephemeral,Book,2024-01-15,adjective,"The ephemeral beauty of spring","nature,literature"
ubiquitous,Article,2024-02-20,adjective,,"technology"
